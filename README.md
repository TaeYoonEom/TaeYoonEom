## 🚀 Key Project & Experience

### 📄 국제 저명학술지 연구 동향 분석 시스템 (졸업 작품)
> **Nginx + Gunicorn + Django 조합의 웹 아키텍처를 설계하고, 정밀 검색 및 AI 연동 데이터 파이프라인을 전담하여 개발한 웹 서비스입니다.**

*   **Architecture & Pipeline:** 
    *   Team 협업을 통한 Nginx, Gunicorn, Django 기반 웹 인프라 공동 설계 및 구축
    *   TF-IDF 및 파이썬 KeyBERT 라이브러리를 활용한 정밀 키워드 검색 및 문맥 분석 기능 구현
    *   오픈소스 LLaMA 모델을 연동하여 다차원 시각화 정보 기반의 정성적 분석 데이터 파이프라인 구축
*   **Trouble Shooting (DB 성능 최적화):**
    *   **문제:** 대량의 논문 데이터 삽입 및 다중 조건 검색 시, 테이블 조인 부하로 인한 심각한 페이지 로딩 지연 발생
    *   **원인:** 프론트엔드의 비동기(Ajax) 처리만으로는 근본적인 데이터베이스 서버 부하 해결 불가 판단
    *   **해결 및 성과:** 테이블 구조 반정규화를 과감히 도입하고, 복잡한 필터링 로직을 데이터베이스단에서 직접 처리하도록 SQL 쿼리를 최적화하여 **응답 속도 획기적 개선 및 시스템 성능 향상**
